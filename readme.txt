Fancy Slider with Desktop Markup to show Websites Portfolios or Vertical Images.
== Description ==

WN Webpage Slider for Elementor, is an addon for Elementor Page Builder Plugin, which allows you to select three images and show them within a desktop markup with a fancy scroll and slide to the right effect. From the back-end you will be able to set some parameters as width, height, horizontal separation, vertical separartion, delay after scroll, slide velocity among others. 

# Free Features

-	Fancy Porfolio Slider
-	Automatic Scroll that shows the entire Image (Website Home Page)
-	Custom Controls on width, speed, etc
-	Responsive Design
-	Live Updated Changes
-	Draggable Widget

https://youtu.be/O9aM1TDeK9g

# How to use it

-	Go to Elementor's Back-end Panel and Search for "Webpages Slider" on the Widgets Basic area.
-	Drag the "Webpages Slider" Widget wherever you want to use it on your webpage.
-	Once you release the widget, the option will appear on the Elementor's Panel as with any other Elementor's widget.
-	Play around and have fun with it :)

== Installation ==

1. In case you don't have Elementor installed, proceed to install Elementor First.
2. Upload the plugin folder 'wn-flipbox-pro' to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
3. Activate the plugin through the 'Plugins' screen in WordPress.
